## General info

Simple web app to play battleships with other players.

## Technologies

* Python
* Django
* JavaScript
  

## Usage 

first install all requirements using:
```console
pip install -r requirements.txt
```
then run 
```console
./manage.py runserver
```
which starts application that can be accessed on 'localhost:8000/battleships'
